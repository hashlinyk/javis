# 知识库索引

知识库是记忆系统的快速检索入口，包含所有知识的索引、标签系统和主题分类。

## 目录结构

```
knowledge_base/
├── README.md          # 本文件
├── index.md           # 知识库主索引
└── tags.md            # 标签系统
```

## 使用说明

### 快速查找

- 查看 `index.md` 获取按领域分类的知识列表
- 使用 `tags.md` 按标签查找相关知识

### 添加新知识

1. 将知识记录到对应的 `memory/domain/` 子目录
2. 在 `index.md` 中添加索引条目
3. 在 `tags.md` 中添加相关标签

## 维护规则

- 每次添加新知识后更新索引
- 保持标签系统的有序性
- 定期整理和归档旧知识
