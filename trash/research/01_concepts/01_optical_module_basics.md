# 光模块基础知识

## 定义

光模块(Optical Module)是光纤通信系统中的核心组件，负责**电信号与光信号之间的相互转换**。

- **地位**: 光纤通信系统的"心脏"
- **入选**: 2023年十大科技热词
- **本质**: "电-光翻译官"

## 工作原理

### 核心功能
```
电信号 → [发射端] → 光信号 → [光纤传输] → 光信号 → [接收端] → 电信号
```

### 核心组件
1. 光发射器 - 将电信号转换为光信号（激光器/LED）
2. 光接收器 - 将光信号还原为电信号（光电探测器）
3. 功能电路 - 信号处理和控制
4. 光接口 - 与光纤连接的物理接口

## 分类体系

### 按速率分类
- **低速**: 155M、622M、1.25G、2.5G
- **中速**: 10G、40G
- **高速**: 100G、200G、400G
- **超高速**: 800G、1.6T（AI数据中心需求）

### 按封装形式分类
- SFP系列: SFP、SFP+、SFP28、SFP56
- QSFP系列: QSFP、QSFP28、QSFP-DD
- CFP系列: CFP、CFP2、CFP4
- 其他: XFP、CXP

### 按距离分类
- 短距(SR): 300m-2km
- 中距(LR/ER): 10km-40km
- 长距(ZR): 80km+

## 应用场景

| 场景 | 需求特点 | 主要速率 |
|------|---------|---------|
| 数据中心 | 高速、高密度、低功耗 | 400G/800G/1.6T |
| 5G基站 | 前传、中传、回传 | 10G/25G/50G |
| 电信骨干 | 超长距离、超大容量 | 100G/200G |
| 企业网络 | 成本敏感 | 1G/10G |

## 关键技术指标

1. **传输速率**: 带宽能力
2. **传输距离**: 无中继传输距离
3. **中心波长**: 850nm/1310nm/1550nm
4. **功耗**: AI数据中心关键指标
5. **成本**: 大规模部署核心考虑

## 2025年技术趋势

1. **速率升级**: 400G→800G→1.6T
2. **功耗优化**: LPO、CPO等技术路线
3. **集成度提升**: 硅光、共封装技术
4. **AI驱动需求**: 数据中心光模块占比持续提升
