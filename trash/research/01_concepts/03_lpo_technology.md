# LPO (Linear Pluggable Optics) 技术详解

## 定义

LPO（线性可插拔光模块）是一种**移除DSP芯片**的创新光模块技术，将数字信号处理功能从光模块转移到交换机侧的ASIC芯片。

## 技术原理

### 核心创新
```
传统光模块: [电信号] → [DSP芯片] → [Driver] → [激光器] → 光信号
LPO方案:     [电信号] → [Driver] → [激光器] → 光信号
                         (DSP移至交换机侧ASIC)
```

### 工作机制
1. **去除DSP芯片** - 仅保留线性驱动器和跨阻放大器（TIA）
2. **线性驱动技术** - 高精度模拟电路直接调制激光器
3. **分段补偿** - ASIC + Driver + TIA协同完成信号补偿
4. **编码支持** - PAM4调制 + FEC前向纠错

### 关键指标对比

| 参数 | 传统DSP方案 | LPO方案 | 优化幅度 |
|------|-------------|---------|---------|
| 400G功耗 | ~12W | <8W | ↓40-50% |
| 800G功耗 | 14-18W | 8-10W | ↓40-45% |
| 信号延迟 | 纳秒级 | 亚纳秒级 | ↓30%+ |
| 成本 | 基准 | -15-20% | 省去DSP芯片 |

## LPO vs CPO 对比

### 架构差异

| 特性 | LPO | CPO |
|------|-----|-----|
| **封装层级** | 模块级（可插拔） | 芯片级（共封装） |
| **传输距离** | ≤2km | ≤500m |
| **DSP处理** | 移至交换机侧 | 完全移除/集成 |
| **功耗(800G)** | 8-10W | 5-7W |
| **成本(800G)** | $1,200-1,500 | $2,000+ |
| **可维护性** | 高（热插拔） | 低（换板卡） |
| **标准化** | OIF推进LRPO | COBO/OpenEye |
| **应用场景** | 机柜间/机柜内 | 机柜内芯片级 |

### 技术定位
```
CPO/NPO: 超短距(<1m) - 芯片级互联
   ↓
LPO: 中短距(1m-2km) - 主流方案 ⭐
   ↓
传统DSP: 长距(>2km) - ER/ZR场景
```

## LPO 的核心优势

### 1. 能效革命 ⚡
- **功耗直降50%**: 400G从12W降至<8W
- **散热成本降低**: 模块降1W → 冷却降0.5W
- **经济效益**: 单机柜100模块年省>$2000

### 2. 极致低延迟 🚀
- 延迟降至亚纳秒级
- 延迟缩减30%+
- AI训练、高频交易场景关键

### 3. 成本优化 💰
- DSP占成本25-30%
- "去DSP化" → BOM降15-20%
- 800G模块降约$50-60

### 4. 兼容性与灵活性 🔧
- 保留QSFP-DD/OSFP封装
- 即插即用
- 无需改造基础设施

### 5. 供应链自主化 🇨🇳
- 减少对海外DSP依赖
- 中国模拟电路设计深厚
- 推动国产化替代

## LPO 的局限性

### 1. 传输距离受限
- 商用产品≤2km（DR场景）
- >10km仍需传统DSP
- 实验室已实现800G LPO 5km，但未商用

### 2. 工艺精度要求高
- 需TEC温漂控制
- 对光芯片线宽、模拟电路精度要求极高
- 生产良率待提升

### 3. 标准化挑战
- IEEE 802.3 LPO标准处于最终草案
- 不同厂商模块有差异
- 需定制化互通测试

## 2025年市场进展

### 商业化进展
- **华工科技**: 800G LPO海外交付，2026年上量
- **英伟达GB200**: 预留LPO接口
- **阿里云**: 张北数据中心启动现网测试
- **市场规模**: 2028年预计突破30亿美元

### 技术演进
1. **硅光集成** - 推动距离向5km迈进
2. **AI智能补偿** - 实时调整激光器偏置
3. **混合架构** - 与CPO梯度覆盖

### 应用场景
- **短期(1-2年)**: 超算中心、AI训练集群、边缘计算
- **长期(3-5年)**: 替代数据中心50%中短距传统模块

## 三种技术路线总结

| 技术路线 | 最佳场景 | 主要优势 | 主要挑战 |
|---------|---------|---------|---------|
| **传统DSP** | 长距(>2km) | 成熟稳定、传输距离远 | 功耗高、延迟大 |
| **LPO** | 中短距(≤2km) | 功耗低、可插拔、成本低 | 距离受限、标准未完善 |
| **CPO** | 超短距(<1m) | 功耗最低、密度最高 | 成本高、可维护性差 |

## 关键结论

1. **LPO是中短距场景的核心方案**
2. **LPO vs CPO不是替代关系，而是互补**
3. **2025-2026年是LPO规模化商用的关键期**
4. **新易盛在LPO技术领域具有差异化优势**
