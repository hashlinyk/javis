## 遇到的问题

### 1. 千亿像素 - 服务器503错误

```
HTTP 503 Service Temporarily Unavailable
```

**原因**：服务器可能有频率限制或临时过载

**解决方案**：
- 添加下载延迟
- 使用重试机制
- 分批下载

### 2. Windows编码问题

**问题**：Python脚本在Windows CMD中显示中文乱码

**已解决**：移除特殊Unicode字符，使用纯ASCII

### 3. 资源路径复杂性

- krpano使用动态URL生成
- 需要解析XML配置文件
- 多级分辨率结构复杂
